Course Curriculum: Web Development Basics

Welcome to the Web Development course. In this course, you will learn how to design simple web pages using HTML and basic formatting tags. Special characters example: © 2025 MyCourse ® All Rights Reserved.

Objectives:

  1. To design a simple course web page using HTML.
  2. To practice the use of basic text formatting tags (bold, italic, underline, and special characters).

Course Modules:

  • Introduction to HTML
  • Text Formatting
  • Lists in HTML
  • Hyperlinks
  • Images

Course Content Description:

HTML
The standard language for creating web pages.
CSS
Used for styling and layout of web pages.
JavaScript
Makes web pages interactive and dynamic.

Detailed Syllabus:

  1. HTML Basics
    • Structure of HTML
    • Tags and Attributes
  2. Formatting
    • Bold, Italic, Underline
    • Special Characters

Assignments:

  • Week 1:
    1. Create a basic HTML file
    2. Use text formatting tags
  • Week 2:
    1. Create lists in HTML
    2. Mix ordered and unordered lists
